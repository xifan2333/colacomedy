# 可乐喜剧小程序

## 项目说明

本小程序是为西安脱口秀俱乐部可乐喜剧定制开发。主要功能有：

- 演出信息查询
- 购票取票
- 导航至演出场地

购票取票功能调用互动吧api及小程序

## 文件结构

目前所有的开发文件存于 `miniprogram` 目录,另一目录用于云开发，暂无开发计划
